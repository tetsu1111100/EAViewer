# EAViewer

**EAViewer** 是一款專為軟體工程師打造的現代化 **SQL Server Table Schema 檢索與管理工具**。基於 WPF 與 Fluent UI 設計，提供直覺且高效的視覺化無限畫布，讓工程師能快速查詢資料表結構、進行跨表比對，並透過豐富的顏色標記功能輕鬆梳理與分類欄位資訊。

---

## 🌟 核心功能特色

### 1. 🔍 快速且詳細的 Schema 檢索
- 支援快速輸入 Table Name 後擷取結構 (Schema) 資訊。
- 可清楚展示**主鍵 (Primary Key)**、**自動遞增 (Auto Increment)**、**允許 Null**、**預設值**及**資料型態**等詳細狀態（停留在欄位上會顯示格式化提示資訊 ToolTip）。
- 支援一鍵將「欄位名稱」切換為全小寫顯示，符合部分語言 ORM 變數慣例。

### 2. 🎨 全新升級：21 款馬卡龍色標記系統
- 針對 `欄位名稱`、`說明`、`備註` 等欄位，透過**點擊右鍵**即可呼叫出顏色選單。
- 內建 **21 種不同深淺的絕美馬卡龍色系**（如🌸櫻花粉、🌿薄荷綠、☁️奶油藍等），能清晰進行資料梳理與狀態分類。
- 支援**將背景色儲存至資料庫**，確保顏色設定不隨程式關閉而丟失。

### 3. ♾️ 視覺化無限畫布體驗
- **多視窗展示**：點擊 `Open New` 可以在同一畫布上同時開啟多個不同資料表的 Grid，極大化跨表關聯比對的效率。
- **拖曳平移 (Panning)**：滑鼠點擊畫布空白處不放並且移動，即可如同地圖般平移整個畫布視角（並會出現拖曳游標提示）。
- **獨立視窗操作**：各個 Table Grid 為獨立卡片，支援「**拖曳移動**」與「**動態縮放大小 (Resize)**」。
- **自動置頂 (Z-Index)**：當點擊任何被遮住的表格標題列時，該表格會自動躍升至畫面最上層。
- **邊界防護機制**：具備防呆機制，防止使用者將 Grid 標題列拖曳至畫布外導致無法關閉。

### 4. ⚡ 互動與操作優化
- **進階搜尋過濾**：點擊欄位表頭 (Header) 可彈出欄位內容搜尋視窗。
- **靈活的列選取 (Row Selection)**：
  - 獨立點擊列可單純反白該列並自動取消其他列表。
  - 按住 `Ctrl` 可進行多重取消/選取操作。
  - 點擊表格空白處會自動取消所有選取狀態。
  - 套用高對比藍色反白與白色字體，確保選取的資料清晰可見。
- **快速複製支援**：支援右鍵單獨複製欄位內容或一鍵複製 Table 名稱。

## 🛠️ 技術架構與依賴套件

EAViewer 是一個基於現代 .NET 軟體架構所開發的桌面應用程式：

- **基礎框架**: .NET 8 (Windows Presentation Foundation)
- **架構模式**: 嚴謹的 **MVVM** 架構 (Model-View-ViewModel)
- **設計風格**: 導入 **WPF-UI** 提供 Windows 11 Fluent 視覺風格 (包含 Mica 背板與現代化控制項)。
- **依賴注入**: 透過 `Microsoft.Extensions.DependencyInjection` 處理 Service 邏輯解耦。
- **MVVM 套件**: `CommunityToolkit.Mvvm` (依賴 Source Generator 提升開發效率)。

## 🚀 如何使用

1. 在上方搜尋框中輸入欲查詢的 SQL `Table Name` 並按下 `Enter` 或 `Search`。
2. 開啟資料表後，您可以自由使用滑鼠拖拽標題列調整表格位置，或點擊畫布空白處移動整個畫布視角。
3. 如果需要比對兩張表，請在搜尋框輸入新的名字後，點擊 `Open New`。
4. 在任何需要標記的欄位上點擊右鍵，從 21 色選單中挑選喜愛的顏色，標記完畢後可點選表格右下角的 `Save Background Color` 進行儲存。
